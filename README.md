# Macklinu Tap

## How do I install these formulae?

`brew install macklinu/tap/<formula>`

Or `brew tap macklinu/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "macklinu/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
